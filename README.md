# Pinned Tabs Restorer

Chrome拡張機能：ピン留めしたタブを自動的に保存し、Chromeを再起動した際に復元します。

## 機能

- ピン留めしたタブのURLを自動保存
- Chrome起動時に保存されたピン留めタブを自動復元
- タブの追加・削除・移動時にリアルタイムで保存

## インストール方法

1. Chromeで `chrome://extensions` を開く
2. 右上の「デベロッパーモード」を有効にする
3. 「パッケージ化されていない拡張機能を読み込む」をクリック
4. このディレクトリを選択

## 使い方

1. タブを右クリックして「ピン留め」を選択
2. Chromeを完全に終了（Mac: Cmd+Q）
3. Chromeを再起動すると、ピン留めしたタブが自動的に復元されます

## 注意事項

Chromeの「前回開いていたページを開く」設定と併用すると、タブが重複する可能性があります。
この拡張機能は、Chromeのネイティブ復元機能でピン留めタブが復元されない場合に特に有用です。
